# Build assets

Эта папка содержит ресурсы для упаковки в инсталлятор.

## Иконки

`icon.svg` — мастер-исходник в фирменной палитре Алисы. Перед сборкой
production-инсталлятора сконвертируйте его в платформенные форматы:

```bash
# PNG (Linux)        → icon.png   1024×1024
# ICO (Windows NSIS) → icon.ico   мульти-размер 16/32/48/64/128/256
# ICNS (macOS DMG)   → icon.icns  16/32/64/128/256/512/1024 (с retina)
```

Утилита `@electron-forge/sharp` или сторонние конвертеры (ImageMagick, Inkscape +
`png2ico` / `iconutil`) справятся за один проход. Готовые файлы кладутся
в эту же папку — `electron-builder` подхватит их автоматически.

Если иконок нет, electron-builder соберёт инсталлятор с дефолтной электрон-иконкой —
работать будет, но вид будет generic.
